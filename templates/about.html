About

{{ title }} is a small, self-hosted URL shortener written in Common Lisp. Nothing here needs JavaScript: w3m and eww get the same pages a graphical browser gets. The one exception is the copy shortcut on a freshly smashed link, which is added by a script when there is one to run and simply absent otherwise.

Command line

Add raw to the register POST and the response is the shortened link as plain text. Handy from the shell: 

curl -sX POST -d url=https://example.com '{{ base-url }}register?raw'

An unusable URL answers with a 400 and a plain-text reason.
